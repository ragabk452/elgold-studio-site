# ELGOLD STUDIO

A premium, bilingual (English / Arabic) one‑page portfolio site for **ELGOLD STUDIO** — a personal design + development studio led by **Kareem Ragab**, specialising in SaaS platforms, CRM systems and dashboards.

## Features
- **Cinematic loader** + full‑page **snap scroll** (cover transitions).
- **Bilingual EN / AR** with full RTL support (toggle, `localStorage` persisted).
- **Real projects** with in‑page **case‑study modals** (cover, problem/solution/results, tech, gallery, live link).
- **Studio / About** section, services, process, tech stack, testimonials, FAQ.
- **Contact form** → WhatsApp, plus direct **PayPal (card)** and **InstaPay** payment.
- Performance‑tuned (optimised images, GPU‑friendly effects, reduced‑motion support, mobile‑light mode).

## Tech
Vanilla **HTML5 / CSS3 / JavaScript** · **GSAP** + ScrollTrigger · **Lenis** smooth scroll. No build step.

## Run locally
```bash
python3 -m http.server 8080
# then open http://localhost:8080
```

## Structure
```
index.html          # the page
css/                # tokens · style · responsive
js/main.js          # loader · i18n · snap scroll · modal · form
assets/             # fonts, project screenshots, founder photo
```
